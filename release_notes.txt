Libnymphcast Release notes

---

> v0.2.1

New:
- Added receiver shares support.

Bugs:
- Fixed installation for Haiku.
- Fixed compilation for FreeBSD.
- Added double quotes for MSVC scripts to handle spaces in paths.


> v0.2

New:
- C binding.
- Haiku platform support.
- Block size parameter for read and seek callbacks.
- Use MSYS2 platform defined by the active shell (Mingw64, UCRT, etc.).

Bugs:
- Add ext-numeric-literals flag to deal with newer GCC versions.


> v0.1

- Initial version.
